XAI Sentinel: Explainable AI for Patient Safety

An advanced Clinical Decision Support System (CDSS) that utilizes XGBoost for risk prediction and SHAP/LIME for visual explainability.

Architecture

Frontend: React.js (Vite) + Tailwind CSS + Lucide Icons.

Backend: Python FastAPI + XGBoost + SHAP.

Database: Firebase Firestore.

Setup

Backend:

pip install -r requirements.txt

python main.py

Frontend:

npm install

npm run dev

Deployment

Frontend is optimized for Vercel.

Backend is optimized for Render.
